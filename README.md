# Fake Job Detection using NLP & Machine Learning

## Project Overview

This project uses Natural Language Processing (NLP) and Machine Learning techniques to identify fraudulent job postings. The model analyzes job descriptions and predicts whether a job advertisement is genuine or fake.

## Problem Statement

Online job portals often contain fake job advertisements that can mislead job seekers. The objective of this project is to build a machine learning model capable of detecting fraudulent job postings automatically.

## Technologies Used

* Python
* Pandas
* NumPy
* NLTK
* Scikit-learn
* Matplotlib
* Seaborn

## NLP Techniques

* Text Cleaning
* Tokenization
* Stopword Removal
* Stemming
* Lemmatization
* TF-IDF Vectorization

## Machine Learning Models

* Logistic Regression
* Naive Bayes
* Linear SVM
* XGBoost

## Results

* Accuracy: **98.6%**
* Best Model: **Linear SVM**

## Key Features

* Automated fake job detection
* NLP-based text preprocessing
* Machine learning classification
* High prediction accuracy
